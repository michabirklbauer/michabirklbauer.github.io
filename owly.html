Owly Messenger

A chatbot written in python

Posted by Micha Birklbauer on December 10, 2018

Introduction

Even though Owly has it's own project page I still wanted to showcase it here since it is my biggest project yet. It is a very simple if-else chatbot that uses phrases said by my friends. Owly Messenger was written completely in python and uses the tkinter package to display a graphical user interface. For a more detailed description read down below!

Owly Messenger V2

The graphical user interface of Owly

Owly Messenger v2 is a program written in python based on Owly Messenger (v1), a C++ application I wrote in 2016. Simply said, it's a chatbot, so it simulates being a person and gives you answers based on your input. These answers vary between the persons it tries to simulate. All answers are based on real-life conversations (and/or situations) that I had with these respectable individuals.

How do I get started?

  • First of all you should read the documentation!
  • Or at least the quick start guide!
  • Download the application in the “Download” section of the official project page!
  • Watch the youtube tutorials on how to install python and run python scripts if you are not familiar with it:
    • Windows: https://youtu.be/nx4c5mie9aw
    • macOS: https://youtu.be/2W5K4Byk_FI
  • Enjoy chatting!
