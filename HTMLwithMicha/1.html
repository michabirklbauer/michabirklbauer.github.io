HTML mit Micha - Teil I

Die Grundidee von HTML ist, dass eigentlich alles aus sogenannten Tags besteht, also aus diesen komischen Elementen bestehend aus "<" und ">" Zeichen. Meistens ist es so, dass es sowohl einen Opening- als auch einen Closing-Tag gibt. Zum Beispiel wird ein HTML-Dokument immer mit "<html>" begonnen und mit "</html>" abgeschlossen. Opening- und Closing-Tag unterscheiden sich also nur durch den Forwardslash.

Grundsätzlich besteht jedes HTML-Dokument außerdem aus einem "header" und "body" Element. Allgemein könnte man diese Struktur schon einmal so zusammenfassen:

Damit haben wir eigentlich schonmal das Grundgerüst für unsere Website. Bleibt uns nur noch das Ganze mit Inhalten zu füllen, die wichtigsten Tags sind hier meiner Meinung nach:

  • Das <title> Element für den Titel der Website - normalerweise im Header.
  • Das <h1> Element für Überschriften (oder auch h2, h3, usw. - je größer die Zahl, desto kleiner die Überschrift).
  • Das <p> Element um einen Textabschnitt zu definieren (Paragraph).
  • Das <a> Element für Links.
  • Das <img> Element für Bilder.
